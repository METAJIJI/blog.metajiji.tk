zabbix in linux

Problem:

root@host# LC_ALL=C su -m zabbix -c "ping ya.ru"
ping: icmp open socket: Operation not permitted
Solution:
sudo chmod u+s `which ping`

Problem:

sudo: sorry, you must have a tty to run sudo
