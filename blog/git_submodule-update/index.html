Обновление подмодулей git

Date Сб. 10 Декабрь 2016 Modified Сб. 10 Декабрь 2016 By Denis Kadyshev Category VCS Tags git / bash

Инициализуруются подмодули так:

git submodule init

Синхронизируются и обновляются так:

git submodule sync
git submodule update

Получить последние версии кода:

git submodule update --recursive --remote
