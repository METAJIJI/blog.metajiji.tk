Enable ip forwarding in linux

Date Пн. 20 Март 2017 Modified Пн. 20 Март 2017 By Denis Kadyshev Category Linux Tags Linux / Centos / Ubuntu / ip_forwaring

Debian/Ubuntu /etc/network/options:

ip_forward=yes

For restart network service procps:

/etc/init.d/procps.sh restart

RedHat /etc/sysconfig/network:

FORWARD_IPV4=true
